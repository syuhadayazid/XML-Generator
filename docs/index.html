XML and EDI Generator

Upload TXT, XLSX, or EDI/X12 mapping input, generate XML in the browser, and generate X12 EDI when X12 paths are detected.

Input

Supported input formats:

  • TXT: one XML path per line
  • XLSX: paths read from one selected column
  • EDI/X12: segments are converted to XML-like path lines

Output

The generator always produces XML. For X12 mappings, it also produces EDI so both outputs can be downloaded.

Works On GitHub Pages

This UI runs fully in the browser, so users can upload a file and generate XML without running PowerShell on the website.
