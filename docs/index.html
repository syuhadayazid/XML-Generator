XML Generator

This PowerShell tool builds a sample XML document from XML path definitions stored in a TXT or XLSX file.

Download Script Download Backup

Input

Supports TXT and XLSX input files that contain XML paths.

  • TXT: one XML path per line
  • XLSX: read XML paths from a selected column

Output

Creates a sample XML file using the root and node structure detected from the paths in the source file.

Requirements

Run locally with PowerShell. XLSX input also requires the ImportExcel module.

How To Use

  1. Download generate_sample_xml.ps1.
  2. Open the script and update the top settings such as inputPath, outputPath, xlsxPathColumn, and xlsxWorksheetName.
  3. If you use XLSX input, install the Excel module once:
Install-Module ImportExcel -Scope CurrentUser
  1. Run the script in PowerShell:
pwsh -File "generate_sample_xml.ps1"

Notes

  • The script auto-detects the XML root from the first valid path.
  • Root namespace declarations can be preserved on the root element.
  • Common malformed path patterns are normalized automatically.
  • Duplicate Party nodes with the same typeCode are merged.
